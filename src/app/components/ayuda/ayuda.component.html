Preguntas Frecuentes

Como hacer un pedido?
  • Encuentre el producto que busca en el listado principal o utilizando el buscador en la barra superior

    NOTA: El peso de los productos es aproximado

  • Presione el boton para agregar al carrito de compras una pieza completa o el boton para agregar media pieza.

    NOTA: este ultimo boton solo esta disponible en los productos que se venden por media pieza

  • Una vez agregados al carrito todos los productos, pulse el carrito que se encuentra en la barra superior.
  • En la siguiente pagina puede modificar lo seleccionado, pulsando
    • para aumentar media pieza o pieza completa (segun el caso)
    • para disminuir media pieza o pieza completa (segun el caso)
    • para eliminar el producto de la lista
  • Una vez revisados los productos, seleccione el lugar de envio y pulse el boton

    NOTA: los precios seran confirmados luego de enviar el pedido

  • Visualizado el costo (incluido envio), pulse el boton para comenzar el envio por Whatsapp.
  • Si esta en la pagina desde un telefono celular se redireccionara a Whatsapp para realizar el envio del mensaje. Si por el contrario se accede a la pagina desde una computadora sera redireccionado a la pagina web donde debera pulsar el boton ENVIAR.
